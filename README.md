# qyro-cilent
A cheat cilent for rd-multiplayer.


Install Gradle from here: https://services.gradle.org/distributions/gradle-8.8-bin.zip

Install Git from here: https://git-scm.com/install/

DO NOT EXTRACT THE ZIP.

To use this cilent, run these commands by order.

git clone https://github.com/9aze/rd-multiplayer

cd rd-multiplayer

tar -xf %USERPROFILE%\Downloads\qyro-cilent.zip -C C:\Users\YOURNAME

gradlew.bat buildClient

gradlew.bat runClient


When rd-multiplayer updates, use these commands in order.

cd C:\Users\YOURNAME\rd-multiplayer

git stash

git pull

tar -xf %USERPROFILE%\Downloads\qyro-cilent.zip -C C:\Users\YOURNAME

gradlew.bat buildClient

gradlew.bat runClient

Have fun with this cilent. Contribute  to it if youd like.
